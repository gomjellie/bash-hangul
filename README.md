# bash-hangul
쉘에 입력하는 한글을 영어로 바꿔줍니다. ex) !ㅎ->!g, ㅣㄴ->ls

# INSTALLATION

## 코드를 클론합니다.

```sh
git clone https://github.com/gomjellie/bash-hangul ~/bash-hangul
```

## ~/.bashrc 의 뒤에 bash-hangul 을 로드하는 코드를 추가해줍니다.


```sh

echo "source ~/bash-hangul/bash-hangul.plugin.sh" >> ~/.bashrc

```

## load ~/.bashrc

```sh
source ~/.bashrc
```
